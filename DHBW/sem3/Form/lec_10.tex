\subsection{NP Vollständig}

Ist ein teil von NP.

Das SAT Problem ist NP vollständig. SAT ist die Entscheidbarkeit von Logischen Formeln (Solver e).

Wir können die verschiedenen NP vollständige Probleme in einander Reduzieren.

So kann das 3-Sat Problem auf das Clique Problem Reduziert werden, indem aus jeder Klausel eine seite im Graph wird. Jeder Konten der Seiten wird mit allen Knoten der anderen Seiten verbunden, wenn diese sich nicht wiedersprechen.
