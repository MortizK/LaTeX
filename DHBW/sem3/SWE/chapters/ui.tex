\section{Benutzeroberfläche}

\subsection{Allgemeines UI-Konzept}

Die Benutzeroberfläche von \textbf{genExam} ist in zwei Haupt-Tabs gegliedert:

\begin{itemize}
    \item \textbf{XML-Tab:} Verwaltung der Exam-Struktur.
    \item \textbf{PDF-Tab:} PDF-Konfiguration und Vorschau.
\end{itemize}

\textbf{XML-Tab:} Wird als \textbf{Split-Design} umgesetzt:

\begin{itemize}
    \item Links: TreeView mit Exam-Struktur (Exam $>$ Task $>$ Subtask)
    \item Rechts: Detail-Pane für das ausgewählte Objekt
\end{itemize}

\subsection{Komponentenübersicht}

\begin{itemize}
    \item \textbf{TreeView-Komponente:} ExamTreeView + TreeNodeContextMenu
    \item \textbf{Detail-Pane-Komponenten:} BreadcrumbBar, AttributeForm, ChildObjectTable, CreateChildButton
    \item \textbf{PopUps:} EditDialog, DeleteConfirmationDialog für jede Ebene
    \item \textbf{PDF-Komponenten:} PDFConfigPane, PDFPreviewPane, PDFActionButtons
\end{itemize}

\subsection{Interaktionen / Beziehungen}

\begin{itemize}
    \item TreeView-Auswahl \(\rightarrow\) Detail-Pane aktualisiert
    \item Buttons in Detail-Pane \(\rightarrow\) öffnen entsprechende PopUps
    \item PopUps \(\rightarrow\) Änderungen am Model
    \item PDF-Tab Aktionen \(\rightarrow\) Generierung oder Vorschau der PDF
\end{itemize}
