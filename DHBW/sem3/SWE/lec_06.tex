\section{Architekturentwurf und UML}

Der Entwurf dient der Gliederung in überschaubaren Einheiten. So kann eine Webseite in Ihre Hauptunterseiten eingeteilt werden. Es ist somit fast immer eine Baumstruktur.

In Programmen können es der namespace oder das package sein.

Die Pfeilrichtung in UML ist eine "ist eine"-Verbindung.

\begin{equation*}
    Customer \to Person
\end{equation*}

Ein Kunde ist keine Person.
