\begin{enumerate}
    \setcounter{enumi}{60}
    \item DDL, DML, DQL, DCL, TCL
    \item Primärschlüssel, Fremdschlüssel, Integritätsbedingungs Attribute, Relationsschema, Wertebereich
    \item In einem ERM Kann man schon erkennen, ob Anomalien oder Redundanzen entstehen. Zudem bietet die Abstraktion eine Möglichkeit zu Normalisieren und Änderungen einfach vorzunehmen.
    \item Um Daten zu sparen, effizienz
    \item check, not null, default, unique, primary key, foreign key
    \item Bei einem Primärschlüssel wird im Hintergrund noch unique, not null und noch einen Index angelegt.
    \item Zwei Tabellen werden mit einem Fremdschlüssel verbunden. Entweder direkt oder mit einer Verbindungstabelle.
    \item Ein Create mit Verbundenen Tabellen bindet diese beiden aneinander. Die Zweite Tabelle muss bereits existieren.
    \item Alter Table, CHeck Constraints
    \item insert, delete, update
    \item Reihenfolge beachten oder Spaltenname
    \item Zufällig
    \item Copy Befehl (import/ export in pgadmin)
    \item In jeder Skill Spalte steht Test
    \item Integrität soll durch Löschregeln bewahrt bleiben.
    \item Die verschiedenen Löschregeln:
    \begin{enumerate}
        \item CASCADE - Hierarchische Abhängigkeit
        \item Set Null - Wenn ein Datensatz erhalten bleiben soll
        \item Restrict - Wenn etwas noch erhalten bleiben muss (oder: Alles andere außer oben)
    \end{enumerate}
    \item Bei mehreren Löschregeln, können Ketten und Zyklen entstehen, welche viel unerwartet gelöscht wird. Zudem sind SET NULL und NOT NULL Wiedersprüchlich.
    \item Bei Update macht Set Null keinen Sinn, da wir einen Primärschlüssel ändern und diesen niemals woanders dann Null setzen, da wir die Beziehung erhalten wollen.
    \item Eine Relation in der Relationalen Algebra ist eine Teilmenge von Kreuzprodukten (jedes mit jedem) von Attributen.
    \item Eine Operation auf einer Relation ist wieder eine Relation.
    \item Selektion wählt bestimmte Tupel aus (Zeilen einschränken). Bei einer Projektion bestimmte Attribute (Spalten einschränken).
    \item Das Prädikat kann nur wahr oder falsch sein.
    \item Selektion und Projektion sind nicht kommutativ.
    \item Bedeutung von "In Relationaler Algebra gibt es keine Multimengen": Es gibt keine gleichen Tupel.
    \item Bei einer Vereinigung zwei Relationen muss die Attributanzahl/ Schema gleich sein.
    \item Nach dem Kreuzprodukt wird das Schema vereinigt.
\end{enumerate}
