\section{Supervised Learning for Classification}

\subsection{Probabilistic Data Science}

% Ähnlich wie in Statistik

Summenregel und Produktregel

\[
p(X) = \sum_{Y} p(X, Y)
\]

\[
p(X, Y) = p(Y|X)p(X)
\]

Bayes' Theorem:

\[
p(Y|X) = \frac{p(X|Y)p(Y)}{p(X)}
\]

oder auch:

\[
\text{posterior} = \text{likelihood} * \text{prior} / \text{evidence}
\]

Es gibt zwei sichten auf Statistik:

\begin{enumerate}
    \item Frequentist
    \item Bayesian
\end{enumerate}
