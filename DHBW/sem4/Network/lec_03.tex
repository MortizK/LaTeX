\subsection{Feldbus}

Hier ging es Hauptsächlich um Master-Slave System und die verschiedenen duplex:

\begin{enumerate}
    \item Vollduplex (4 Draht Verbindungen)
    \item Halbduplex (2 Draht Verbindungen)
\end{enumerate}

Und noch einen Repeater um mehrere Signale in einem Master-Slave zu verlängern.

\subsubsection*{IO-Link}

Dies ist ein Standard: IEC 61131-9 Single drop Digital Communication Interface for small sensors and actuators SDCI.

Für P2P entwickelt (kein Feldbus)

\subsection{USB - Universal Serial Bus}

Ist die Technologie, welche sich gegen FireWire durchgesetzt hat (obwohl USB die schlechtere Technologie ist)

USB hat aber zum erstem mal Hot Plug eingeführt, braucht aber immer einen Host (den PC)
